# Mini ISO 27001 Gap Assessment Project

## Overview
This project is a portfolio-ready mini ISO 27001:2022 gap assessment for a fictional 25-person SaaS company named NorthstarHR. It is designed to demonstrate scoping, control assessment, findings development, risk prioritization, and remediation planning in a format suitable for LinkedIn, GitHub, and portfolio use.

## Project components
- **Assessment workbook** with five sheets: Overview, Control Matrix, Findings Register, Roadmap, and Questionnaire.
- **Scenario** focused on a cloud-native SaaS environment using AWS, Microsoft 365, endpoints, vendors, and core security operations practices.
- **Assessment approach** using selected clauses and Annex A controls, with maturity scores of 0 (absent), 1 (partial), and 2 (implemented).
