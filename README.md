### Title
Mini ISO 27001 Gap Assessment for a Fictional SaaS Company

### Summary
A sample GRC project that evaluates a fictional SaaS company against selected ISO 27001:2022 requirements and produces a practical remediation roadmap.

## Overview
This is mini ISO 27001:2022 gap assessment for a fictional SaaS company named NorthstarHR. It is designed to demonstrate scoping, control assessment, findings development, risk prioritization, and remediation planning.

## Project components
- **Assessment workbook** with five sheets: Overview, Control Matrix, Findings Register, Roadmap, and Questionnaire.
- **Scenario** focused on a cloud-native SaaS environment using AWS, Microsoft 365, endpoints, vendors, and core security operations practices.
- **Assessment approach** using selected clauses and Annex A controls, with maturity scores of 0 (absent), 1 (partial), and 2 (implemented).

### Skills demonstrated
- ISO 27001 control mapping
- Gap identification and evidence review
- Risk-based prioritization
- Executive-style reporting
- Remediation planning

### Deliverables
- Control matrix
- Findings register
- 90-day roadmap
- Stakeholder questionnaire
